# Welcome to L2J Mobius Chronicle 4

L2J Mobius Chronicle 4 (Scions of Destiny) is a professional-grade Lineage 2 server emulator that brings the classic Chronicle 4 experience to life. Built on a modern Java 25 codebase with over 1,500 source files, this project offers exceptional stability, performance, and authenticity.

## What is Chronicle 4?

Chronicle 4: Scions of Destiny represents the pinnacle of classic Lineage 2 gameplay, introducing:

- **Major Territories**: Varka Silenos Outpost, Ketra Orc Outpost, Imperial Tomb, Hot Springs, Forge of the Gods, and more
- **Noblesse System**: Achieve nobility status and unlock exclusive privileges
- **Heroes & Olympiad**: Compete in ranked PvP battles using an advanced ELO-based matchmaking system
- **Skill Enchanting**: Enhance your abilities beyond their base power
- **Epic Raids**: Challenge Valakas, Antharas, Baium, and other legendary bosses
- **Dimensional Rift**: Explore instanced dungeons with unique rewards
- **Fishing System**: Relaxing fishing mechanics with valuable catches
- **Clan Wars**: Territory control and castle sieges with strategic gameplay

## Chronicle Evolution

This emulator includes features from Chronicle 4 and earlier versions:

### Chronicle 3 Features
- Seven Signs territory control system
- Subclass system for character progression
- Community board mail and friends systems
- Anakim and Lilith raid bosses

### Chronicle 2 Features
- Castle siege mechanics with destroyable walls and traps
- Manor system for clan resource management
- Mini games: Monster Race, Lottery, Dice
- Baium and Zaken raid encounters
- Swimming mechanics
- Boat transportation system

### Chronicle 1 Features
- Castle siege system with 9 castles
- Antharas raid encounter
- Clan halls and clan management
- Weight system mechanics

### Prelude Features
- Complete class transfer system
- Summons and pet mechanics
- Clan and alliance systems
- Player-run shops and trading
